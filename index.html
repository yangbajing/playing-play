Playing Play

Playing Play

玩Play，愉悦的使用Play。

(本书基于Play 2.5，它与以前的版本有稍许不同。请注意更新您的知识库。默认都在Mac或Linux/Unix下开发，使用Windows的同学请注意可能的不同之处)

Play官方对自己的介绍是＂The High Velocity Web Framework For Java and Scala＂，滋以为是符合的。

本地读阅

git clone https://github.com/yangbajing/playing-play.git
cd playing-play
npm -g install gitbook-cli
gitbook install
gitbook serve

打开浏览器访问：http://localhost:4000
